用autohotkey管理windows

功能：
- 快速移动窗口
- 模式键 可定义各种模式（需要自行配置）
- 预定义程序的输入法
- 程序切换语音和图标提示

* 基本命令
| 快捷键           | 动作                               |
|------------------+------------------------------------|
| Ctrl + Alt + F11 | 编辑配置文件                       |
| Ctrl + Alt + F12 | 重新加载配置                       |
| Ctrl + 1         | AppsKey,快捷键只在 everything 生效 |
|------------------+------------------------------------|


* 窗口移动模式
启动快捷键 Win + o, 默认显示器为 2x4(2行4列)
| 模式快捷键 | 动作         |
|------------+--------------|
| 1-9        | Goto Grid no |
| c          | Up           |
| t          | Down         |
| h          | Left         |
| n          | Right        |
| g          | Prev Monitor |
| r          | Next Monitor |
| Esc        |              |
| q          |  exit        |
| s          |              |
|------------+--------------|
