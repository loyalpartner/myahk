* 基本命令
| 快捷键           | 动作                               |
|------------------+------------------------------------|
| Ctrl + Alt + F11 | 编辑配置文件                       |
| Ctrl + Alt + F12 | 重新加载配置                       |
| Ctrl + 1         | AppsKey,快捷键只在 everything 生效 |
|------------------+------------------------------------|


* 窗口管理
| 快捷键         | 动作                  |
|----------------+-----------------------|
| Ctrl + Alt + H | 打开emacs             |
| Ctrl + Alt + T | 打开chrome            |
| Ctrl + Alt + N | 打开Windows Terminal  |
| Ctrl + Alt + S | 打开资源管理器        |
| Ctrl + Alt + 1 | 移动窗口到桌面1       |
| Ctrl + Alt + 2 | 移动窗口到桌面2       |
| Win + 1        | Grid 2x4              |
| Win + 2        | Grid 2x2              |
| Win + 3        | Grid next             |
| Win + 4        | Grid prev             |
| Win + 5        | Grid Center           |
| Win + 6        | Grid Full Screen      |
| Win + 7        | Grid 2x4 7            |
| Win + 8        | Grid 2x4 8            |
| Win + 9        | Load layount          |
| @TODO          | 关闭窗口              |
| @TODO          | 关闭干扰窗口          |
| Ctrl + Alt + , | 置顶窗口              |
| Ctrl + Alt + . | 临时透明窗口          |
| Win + '        | 关闭窗口              |
| Alt + 1        | scroll up in chrome   |
| Alt + 2        | scroll down in chrome |
|----------------+-----------------------|

| Ctrl + n      | down             |
